RegisterType

enum class RegisterType

Represents the type of a register.

Enum Values

Input

A register that contains input values from a data set instance.

Calculation

A register that can be used for calculations and will be loaded with some default value (given by a DefaultValueProvider).

Constant

A register that is read-only and will be initialised with values from a set of constants in the LGP environment.

Unknown

A register that does not fall in some known range will have unknown type. This type primarily serves as a default case when mapping register index ranges to register types.
