Creates a new RegisterSet with the specified parameters.

Parameters

inputRegisters - Number of registers reserved for input values from a data set instance.

calculationRegisters - Number of registers reserved for calculations (i.e. free registers).

constants - A collection of constants to be stored in read-only constant registers.

defaultValueProvider - An implementation that can provide default values for the calculation registers.
