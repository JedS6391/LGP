writeInstance

fun writeInstance(data: Sample<T>): Unit

Writes an instance from some data set to the register sets input registers.

NOTE: The number of input registers of the set must match the number of features in the instance.

Parameters

data - An instance to write to the input registers.

Exceptions

RegisterWriteRangeException - When the number of features does not match the number of input registers.
