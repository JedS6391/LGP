apply

abstract fun apply(index: Int, modifier: (TData) -> TData): Unit

Applies the given modifier to the value of the Register at index.

Parameters

index - The register to modify the value of.

modifier - A function that modifies the value in the register.

Exceptions

RegisterReadException - When index is out-of-bounds of the register set.
