inline fun <TModule : Module> instance(type: RegisteredModuleType): TModule
Content copied to clipboard

Provides an instance of the Module registered for the given type.

The module will be loaded and cast to the type given if possible to cast to that type. This method is able to perform safe casts due to Kotlin's reified generics. If the cast can't be done safely, a ModuleCastException will be thrown and contains details about the failed cast. Due to the use of reified generics, callers of this function can be assured that a ClassCastException won't occur.

NOTE: This function cannot be used from Java; use the instanceUnsafe function instead.

Return

An instance of the module registered for the given module type cast safely as TModule.

Parameters

type

The type of of module to get an instance of.

TModule

The type to cast the module as.

Throws

nz.co.jedsimson.lgp.core.modules.MissingModuleException

When no builder has been registered for the type of module requested.

nz.co.jedsimson.lgp.core.modules.ModuleCastException

When the requested module can't be cast to the type given.
