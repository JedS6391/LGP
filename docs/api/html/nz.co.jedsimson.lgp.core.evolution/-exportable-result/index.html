ExportableResult

interface ExportableResult<T>

Represents a result that is able to be exported from the system.

The type of result that it represents is flexible and will depend on which part of the system it comes from. The idea of being exportable means that the internal representation can be whatever, as long as it has a way to be exported to a list of string pairs.
