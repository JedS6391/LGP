ProgramGenerator(environment: Environment<TProgram, TOutput>, instructionGenerator: InstructionGenerator<TProgram, TOutput>)

Generates Program instances to be used in an LGP population.

The implementation is done as a module, allowing for custom generation logic for programs to be defined. The only requirement is that calls to the next method returns a sequence of program samples.

An implementation is expected to provide an Environment, in order to provide access to any component loaded into the the LGP environment, as well as an InstructionGenerator. Instruction generators themselves are modules, so each can be defined as a module.
