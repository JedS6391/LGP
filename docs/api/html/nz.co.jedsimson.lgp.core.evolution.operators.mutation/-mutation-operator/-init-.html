MutationOperator(environment: EnvironmentFacade<TProgram, TOutput, TTarget>)

A search operator used during evolution to mutate an individual from a population.

The individual is mutated in place, that is a call to MutationOperator.mutate will directly modify the given individual.

Parameters

TProgram - The type of programs being mutated.

TOutput - The type of the program output(s).
