IslandMigration(environment: Environment<TProgram>, options: IslandMigrationOptions)

A model for evolution using a island-migration algorithm.

In an island-migration algorithm, the population is split into a number of islands, and a fixed number of solutions migrate between the islands with some interval.

This is done in an attempt to promote diversity in the population to prevent early convergence on local optima.

Parameters

environment - The environment that evolution is taking place in.

options - Determines the configuration for the algorithm. See IslandMigrationOptions for more.
