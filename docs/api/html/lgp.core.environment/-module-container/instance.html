instance

inline fun <reified TModule : Module> instance(type: RegisteredModuleType): TModule

Provides an instance of the module type given.

The module will be loaded and cast to the type given if possible to cast to that type. This method is able to perform safe casts due to Kotlin's reified generics. If the cast can't be done safely, a ModuleCastException will be thrown and contains details about the failed cast. Due to the use of reified generics, callers of this function can be assured that a ClassCastException won't occur.

NOTE: This function cannot be used from Java; use the instanceUnsafe function instead.

Parameters

type - The type of of module to get an instance of.

TModule - The type to cast the module as.

Exceptions

MissingModuleException - When no builder has been registered for the type of module requested.

ModuleCastException - When the requested module can't be cast to the type given.

Return
An instance of the module registered for the given module type.
