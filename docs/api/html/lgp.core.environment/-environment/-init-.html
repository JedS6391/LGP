Environment(configLoader: ConfigurationLoader, constantLoader: ConstantLoader<TProgram>, operationLoader: OperationLoader<TProgram>, defaultValueProvider: DefaultValueProvider<TProgram>, fitnessFunctionProvider: FitnessFunctionProvider<TProgram, TOutput>, resultAggregator: ResultAggregator<TProgram>? = null, randomStateSeed: Long? = null)

Builds an environment with the specified construction components.

Parameters

configLoader - A component that can load configuration information.

constantLoader - A component that can load constants.

operationLoader - A component that can load operations for the LGP system.

defaultValueProvider - A component that provides default values for the registers in the register set.

fitnessFunctionProvider - A function used to evaluate the fitness of LGP programs.

randomStateSeed - Sets the seed of the random number generator. If a value is given, the seed will be set, and will produce deterministic runs. If null is given, a random seed will be chosen.
