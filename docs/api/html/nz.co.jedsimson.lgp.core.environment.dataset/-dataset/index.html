Dataset

class Dataset<TData, TTarget : Target<TData>>

A basic data set composed of a vector of input Samples and a collection of output Targets.

NOTE: The type of the inputs and outputs is constrained to be the same.

Parameters

TData - The type of the features in the input vector.

TTarget - The type of the outputs in the input vector.
