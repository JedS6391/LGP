DatasetLoader

interface DatasetLoader<TData, TTarget : Target<TData>> : ComponentLoader<Dataset<TData, TTarget>>

An extended ComponentLoader that loads a Dataset.

The source of the data set is controllable by any implementor, all that is required is that the raw data is gathered into a Dataset instance, while satisfying the type constraint TData for individual features in the data set.

Parameters

TData - Type of features in the data set.

TTarget - Type of targets in the data set.
