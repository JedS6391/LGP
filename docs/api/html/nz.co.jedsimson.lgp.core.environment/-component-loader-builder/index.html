ComponentLoaderBuilder

interface ComponentLoaderBuilder<out TComponentLoader>

A builder that can build a ComponentLoader.

Primarily used when creating an implementation of ComponentLoader so that the loader provides a simple way to be built, particularly when used through Java. In Kotlin, it is preferred to use named arguments to the constructor of the loader.
