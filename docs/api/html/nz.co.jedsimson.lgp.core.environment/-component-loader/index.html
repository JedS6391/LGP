ComponentLoader

interface ComponentLoader<out TComponent> : Module

A Module that is able to load components.

The implementer controls how a component the loader is responsible for is loaded and consumers should be indifferent to the implementation details.

Parameters

TComponent - The type of component that this loader is able to load.
