registeredModuleUnsafe

fun <TModule : Module> registeredModuleUnsafe(type: RegisteredModuleType): TModule

Similar to registeredModule, but may give a ClassCastException at the call site.

The internals of the module retrieval system cannot guarantee that any casting will be done safely, and thus this method should be used with caution. The reason for its existence is to make life easier for those using the API through Java.

Parameters

type - The type of registered module to fetch.

TModule - The type the module will be cast as.

Exceptions

MissingModuleException - When no builder has been registered for the type of module requested.

Return
An instance of the module registered for the given module type.
