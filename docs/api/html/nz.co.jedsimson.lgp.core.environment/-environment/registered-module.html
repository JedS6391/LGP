registeredModule

inline fun <reified TModule : Module> registeredModule(type: RegisteredModuleType): TModule

Fetches an instance of the module registered for a particular module type.

The environment assumes that a module type will have a builder registered to if it is being requested. If this fails, then a MissingModuleException will be thrown, indicating that an instance of a particular module type was requested but could not fulfilled.

Usually, the type parameter needn't be explicitly given, as it can be inferred from the type of value the result is being assigned to.

Parameters

type - The type of registered module to fetch.

TModule - The type the module will be cast as.

Exceptions

MissingModuleException - When no builder has been registered for the type of module requested.

Return
An instance of the module registered for the given module type.
