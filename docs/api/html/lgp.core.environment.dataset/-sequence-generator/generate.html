generate

fun generate(start: Double, end: Double, step: Double, inclusive: Boolean = false): Sequence<Double>

Generates a sequence of values between start and end, with an interval of step between each value.

If inclusive is true, then the range will include the end point (i.e. start,end); otherwise, the range will be exclusive (i.e. [start, end))

Parameters

start - The starting point of the range of values.

end - The ending point of the range of values.

step - The interval between each value in the sequence.

inclusive - When true, the range will include end.
