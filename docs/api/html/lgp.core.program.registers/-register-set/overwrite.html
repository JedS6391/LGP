overwrite

fun overwrite(index: Int, value: T): Unit

Forcefully sets the value or the register with the given index.

This method has similar functionality to the RegisterSetset method but does not check the type of the register -- allowing constant registers to be overwritten without exception.

Parameters

index - The register to write to.

value - The value to write to the register.
