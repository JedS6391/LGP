RunBasedExportableResult(run: Int, statistics: EvolutionStatistics)

An ExportableResult implementation that represents evolution statistics for a particular run.

Because evolution statistics are collected on a per-generation basis, this type of exportable result collects data on each generation for each run.

Parameters

run - The run this result relates to.

statistics - Evolution statistics for a particular generation.
