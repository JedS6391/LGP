execute

open fun execute(arguments: Arguments<TData>): TData

Executes an operation in some way.

A base implementation is provided which:

  • Validates that the correct number of arguments is provided.
  • Applies function to the given arguments.

This function is marked as open to allow extension if the base implementation does not provide the behaviour necessary for other use cases.

Parameters

arguments - A set of arguments to the function.

Exceptions

ArityException - When the number of arguments does not match the arity.

Return
A concrete value of type TData.
