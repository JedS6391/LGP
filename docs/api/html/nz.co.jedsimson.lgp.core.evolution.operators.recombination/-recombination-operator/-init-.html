RecombinationOperator(environment: EnvironmentFacade<TProgram, TOutput, TTarget>)

A search operator used during evolution to combine two individuals from a population.

The individuals are mutated directly by the reference given, so calls to this function directly modify the arguments.

Parameters

TProgram - The type of programs being combined.

TOutput - The type of the program output(s).
