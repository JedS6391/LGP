An implementation of ConstantLoader that can load raw constants into a specified type.

The type parameter controls what type the raw constants should be converted into, using a suitable parsing function. This function will be used to map raw constant values to a value in the domain of the type parameter.

NOTE: This implementation caches the results of loading constants, because the loader is immutable - that is, once it is created it can not be altered to load a different set of constants. This is to ensure that the set of constants does not change within an LGP run.
