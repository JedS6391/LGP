Class BaseProblem

public final class BaseProblem
extends Problem<Double, Outputs.Single<Double>, Targets.Single<Double>>

    A basic skeleton for a problem setup with commonly used components and modules.

    This class is supposed to streamline the process of setting up a problem by providing a set of reasonable defaults and a base set of parameters that can be tweaked.
