Class GenericConstantLoader.Builder

  • All Implemented Interfaces:
    nz.co.jedsimson.lgp.core.environment.ComponentLoaderBuilder 
    
public final class GenericConstantLoader.Builder<U extends Object>
 implements ComponentLoaderBuilder<GenericConstantLoader<U>>

    A custom ComponentLoaderBuilder implementation for building a GenericConstantLoader instance.

    The builder allows for a collection of constants (as strings) to be specified, which will be parsed into the type specified as a type parameter.
